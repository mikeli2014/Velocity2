## 关键事实速览

- **演示企业**:虚构的「北海智能家居」(Beihai Smart Home),家电与厨卫电器行业。
- **技术栈**:React 19 + Vite 8 前端,FastAPI + SQLAlchemy 2.0 后端,Anthropic SDK(Sonnet 4.6 + Haiku 4.5)。
- **部署**:统一 Cloud Run 服务(`velocity2`),前后端同一镜像。
- **数据库**:开发用 SQLite,Cloud Run 默认仍是 SQLite(冷启动时由 `auto_seed` 重新填充);切换到 Postgres 仅需设置 `VELOCITY_API_DATABASE_URL` 环境变量,代码已就绪。
- **AI 调用**:全部走 Anthropic 官方 SDK,默认开启 prompt caching;未设置 `ANTHROPIC_API_KEY` 时所有 AI 端点会软退化为 `503 anthropic_not_configured`,前端展示种子内容。
- **测试**:pytest 127 个用例 + Playwright 29 个端到端用例。
- **代码仓**:`mikeli2014/Velocity2`,主干分支 `main`。
